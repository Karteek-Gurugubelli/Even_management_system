About Us

  • The aim of this project is to make and provide an ideal website to book-events, that protects user data and to save the valuable time of the user. Event Management System is a website which provides each and every information of the event like catering, wedding planning and travels. It can be accessible from any place. It makes easy for user to access any event. This system is used to fulfil the requirements of user.

  • In existing system, the person has to visit the particular office for enquiry. This process is quite complex and the user has to spend more time. The Event Management System has to maintain records of the event, it consumes lot of time of user.
